# Cour de Cassation RDC

Ce projet est une application développée avec **TypeScript** et **Prisma** pour la Cour de Cassation de la République Démocratique du Congo.

## Installation

1. Cloner le projet
2. Installer les dépendances : `npm install`
3. Configurer le fichier `.env`
4. Lancer le projet : `npm run dev`
